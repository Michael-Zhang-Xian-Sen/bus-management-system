## 公交管理系统

由两大模块组成，用户模块及公交信息管理模块。

### 1. 用户模块

* 普通用户登录
* 系统用户登录
* 普通用户注册

### 2. 公交信息管理模块

* 查看公交信息
* 增加公交信息
* 修改公交信息
* 删除公交信息
